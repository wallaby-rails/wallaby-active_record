# Wallaby::Core

This gem contains the core interface that Wallaby is built upon.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'wallaby-core'
```

And then execute:

```shell
$ bundle
```

Or install it yourself as:

```shell
$ gem install wallaby-core
```
