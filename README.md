Wallaby::ActiveRecord is the ActiveRecord adapter for [Wallaby::Core](https://github.com/wallaby-rails/wallaby-core).

## Install

Add `Wallaby::ActiveRecord` to `Gemfile`.

```ruby
gem 'wallaby-active_record'
```

And re-bundle.

```shell
bundle install
```
